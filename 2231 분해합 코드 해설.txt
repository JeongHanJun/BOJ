입력된 값 n 
for문을 통해 i = 0~n
각 자릿수로 나눈후 계속 해서 더한다.
sum += num % 10;	// 가장 낮은 자릿수의 값(num%10 즉 num의 1의자릿수)를 sum에 더함
num /= 10;		// num을 자릿수 한칸 낮추기(10진수 num)
i + sum = n 이면 거기서 종료
결국 i = 숫자, sum = 자릿수들의 합 이 되는 것.

올린 코드 2개 중 String 으로 받는것이 4배정도 느림, 비효율적
이유는 count가 0부터~n까지 가는데
그 값을 String으로 받고, char형으로 바꾸고, 다시 int형으로 바꾸므로 
이 값을 받고 변하고 하는 과정이 n이 커질수록 더 오래걸리기 때문
